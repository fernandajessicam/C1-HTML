O que é CSS e para que ele serve?

CSS é a sigla para Cascading Style Sheets, que em português significa "Folhas de Estilo em Cascata". Ele é uma linguagem usada para definir a apresentação visual de páginas da web.

Com o CSS, é possível controlar o layout, cores, fontes, espaçamento, posicionamento dos elementos e diversos outros aspectos visuais de um site. Ele trabalha junto com o HTML, que é responsável pelo conteúdo da página, enquanto o CSS cuida da aparência.

Uma das grandes vantagens do CSS é a separação entre o conteúdo (HTML) e o estilo (CSS), o que facilita a manutenção e reutilização do código. Além disso, ele permite aplicar estilos de forma consistente em várias páginas ao mesmo tempo.

O CSS pode ser aplicado de três formas: diretamente nos elementos HTML (inline), dentro da tag <style> no cabeçalho da página (interna), ou em arquivos separados com extensão .css (externa), sendo essa última a forma mais recomendada.
